# Proyecto Landing Page - Marca Personal

Este proyecto está desarrollado con Django y contiene dos apps:

- `principal`: contiene toda la landing page de marca personal.
- `extra`: app secundaria creada como complemento.

## Cómo ejecutar

1. Instalar Django:

```bash
pip install django
```

2. Entrar a la carpeta del proyecto:

```bash
cd mi_marca_personal
```

3. Ejecutar el servidor:

```bash
python manage.py runserver
```

4. Abrir en el navegador:

```text
http://127.0.0.1:8000/
```

App secundaria:

```text
http://127.0.0.1:8000/extra/
```
